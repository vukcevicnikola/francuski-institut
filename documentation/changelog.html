v3.0.0 –  July 6, 2021

  • New
    Rebuilt Createx core to fully support Bootstrap v5.0.2
    • Dropped jQuery entirely. Redesigned Createx JS to be modular (ES6 modules) and library agnostic.
    • Renamed custom CSS classes to follow the Bootstrap 5 naming convention
    • Rewrote Createx utility classes to support new Bootstrap 5 Utility API
    • No longer supports IE <=11
  • New
    Offcanvas component that replaced the old one
  • Updated
    Node.js to latest version 16x. Switched from deprecated node-sass to dart-sass. Replaced all cases of division / with multiplication * inside SCSS code.
  • Updated
    JavaScript plugins and libraries to latest versions
  • Updated
    Documentation
  • Fixed
    Minor bug fixes and improvements
